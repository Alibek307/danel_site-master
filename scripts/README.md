# SEO Scripts

Автоматическая генерация SEO файлов для многоязычного сайта JasaLab.

## Файлы

### `site-config.js`
Конфигурация сайта со всеми роутами, языками и метаданными:
- Список всех страниц с приоритетами и частотой обновления
- Настройки для разных языков
- Базовый URL и другие глобальные настройки

### `generate-sitemap.js` 
Генерирует `sitemap.xml` с:
- Всеми страницами для всех языков
- Правильными hreflang связями
- Приоритетами и датами обновления
- XML структурой согласно стандартам

### `generate-robots.js`
Генерирует `robots.txt` с:
- Allow записями для всех страниц и языков  
- Группировкой по категориям для читаемости
- Ссылкой на sitemap
- Настройками crawl-delay

### `generate-seo-files.js`
Главный скрипт, который запускает генерацию всех SEO файлов.

## Использование

```bash
# Сгенерировать все SEO файлы
npm run seo:generate

# Сгенерировать только sitemap.xml
npm run seo:sitemap  

# Сгенерировать только robots.txt
npm run seo:robots
```

## Добавление новой страницы

1. Добавьте роут в `site-config.js`:
```javascript
'/new-page': {
  priority: 0.8,
  changefreq: 'monthly', 
  lastmod: '2025-02-08'
}
```

2. Запустите генерацию:
```bash
npm run seo:generate
```

Страница автоматически появится:
- В sitemap.xml для всех языков (EN/RU)
- В robots.txt с правильными Allow записями
- С корректными hreflang связями

## Преимущества

✅ **Автоматизация**: Нет риска забыть добавить страницу для какого-то языка  
✅ **Консистентность**: Одинаковая структура и форматирование  
✅ **Масштабируемость**: Легко добавлять новые языки или страницы  
✅ **Поддержка**: Изменения в одном месте применяются везде  

## Структура вывода

После выполнения команды обновляются файлы:
- `public/sitemap.xml`
- `public/robots.txt`

Не забудьте закоммитить изменения!
